# 🚗 RentalCar — Платформа оренди автомобілів

Фронтенд-частина вебзастосунку для компанії **RentalCar**, яка займається орендою автомобілів. Проєкт реалізовано з використанням Next.js відповідно до технічного завдання.

## 🛠 Стек технологій
- **Фреймворк:** Next.js (App Router)
- **Мова програмування:** TypeScript
- **Управління запитами:** TanStack Query (React Query)
- **Стилізація:** CSS Modules

## 📂 Структура маршрутизації
- `/` — Домашня сторінка з hero-секцією та кнопкою View Catalog.
- `/catalog` — Сторінка каталогу з пагінацією Load More та серверною фільтрацією.
- `/catalog/[carId]` — Сторінка деталей автомобіля з формою оренди та кастомним календарем. Відкривається в новій вкладці.

## ✨ Реалізований функціонал
- **Фільтрація:** Пошук за брендом, ціною та пробігом (із фіксованими приставками From/To в інпутах).
- **Пагінація:** Динамічне довантаження через `useInfiniteQuery`.
- **Кастомний календар:** Повністю робоче перемикання місяців та вибір дати без сторонніх бібліотек.
- **Форма оренди:** Асинхронна відправка даних з лоадером та успішною нотифікацією.

## 💻 Запуск проєкту локально
1. `npm install` — встановити залежності.
2. Створити `.env.local` і додати: `NEXT_PUBLIC_API_URL=твоє_api`
3. `npm run dev` — запуск локально.
